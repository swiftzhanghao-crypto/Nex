+ 查询性别为“男”的学生列表。

```plain
SQL

SELECT name
FROM student__c
WHERE sex__c = '男'
```

student__c 为自定义对象“学生”的 API 名称，性别字段为单选类型字段。

+ 查询爱好选择了足球和游泳的学生列表。

```plain
SQL

SELECT name
FROM student__c
WHERE hobby__c = '足球;游泳'
```

    - student__c 为自定义对象“学生”的 API 名称，爱好字段为多选类型字段。
    - 多个选项值使用英文模式的分号（;）进行分隔。
+ 单选、多选类型字段查询及 IN 的使用。

在标准对象“客户”中有多选类型字段“城市”，客户中共有三条数据，现查询所有数据的城市字段，返回结果为：

北京,南京

南京,天津

北京,石家庄

以下为在 IN 中通过“code”筛选数据的示例：

    - 查询城市包含“北京”的数据（假设“北京”的 code 值为 1）。

```plain
SQL

SELECT city
FROM account
WHERE city IN (1)
```

返回结果为：

北京,南京

北京,石家庄

    - 查询城市包含“北京”和“天津”的数据（假设“北京”的 code 值为 1，“天津”的 code 值为 11）。

```plain
SQL

SELECT city
FROM account
WHERE city IN (1, 11)
```

返回结果为：

北京,南京

南京,天津

北京,石家庄
