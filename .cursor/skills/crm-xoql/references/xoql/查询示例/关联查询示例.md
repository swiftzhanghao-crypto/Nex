## 向上查询示例
+ 查询指定客户的客户联系人的姓名、电话。

```plain
SQL

SELECT id, contactName, phone
FROM contact
WHERE accountId.accountName = '仁科互动'
```

+ 查询指定客户下联系人姓名、订单总金额。

```plain
SQL

SELECT contactName, accountId.accountName, accountId.totalOrderAmount
FROM contact
WHERE accountId.accountName = '仁科互动'
```

+ 查询所有客户以及客户的联系人数量。

```plain
SQL

SELECT COUNT(id), accountId.accountName
FROM contact
GROUP BY accountId.accountName
```

## 向下查询示例
+ 查询指定客户以及客户下的所有联系人。

```plain
SQL

SELECT accountName, (SELECT id, contactName FROM contact WHERE accountId = account.id)
FROM account
WHERE accountName = '仁科互动'
```

+ 查询指定客户以及客户下由张三创建的联系人。

```plain
SQL

SELECT accountName, (SELECT id, contactName FROM contact WHERE accountId = account.id AND createdBy = '991840')
FROM account
WHERE accountName = '仁科互动'
```

## 向上、向下混合查询示例
查询指定客户所有人的姓名以及客户下所有联系人的姓名。

```plain
SQL

SELECT ownerId.name ownerName, accountName, (SELECT contactName FROM contact WHERE accountId = account.id)
FROM account
LIMIT 10
```
