在对单选、多选类型字段进行查询时，可以自定义选项列表值的查询方式，即返回选项内容还是选项 code。

| **参数** | **说明** |
| --- | --- |
| xoql | 查询语句。 |
| useSimpleCode | 单选、多选类型字段选项列表值的返回形式。<br/>true：返回选项列表值的 code 值。<br/><img src="https://cdn.nlark.com/yuque/0/2026/png/26587655/1777434306863-168b9c7c-669c-453a-ad96-50b457f20389.png" width="912" title="" crop="0,0,1,1" id="u3eb7c659" class="ne-image" style="font-size: 14px"><br/>false：返回选项列表值的具体内容。<br/><img src="https://cdn.nlark.com/yuque/0/2026/png/26587655/1777434306865-45ca54ea-e79e-44da-8439-84d906bc4b8c.png" width="908" title="" crop="0,0,1,1" id="ufb62e19c" class="ne-image" style="font-size: 14px"><br/>此项缺省时，默认为 false。 |
