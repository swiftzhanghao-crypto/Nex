一般情况下，当使用 GROUP BY 子句时，需要配合使用聚合函数，但不使用 GROUP BY 子句，也可以使用聚合函数。

## AVG
返回指定字段的平均值。

例如，查询订单对象上，各客户总金额的平均值。

```plain
SQL

SELECT AVG(amount)
FROM _order
GROUP BY accountId
```

+ 支持的字段类型请参考[支持在聚合函数中使用的字段类型](https://doc.xiaoshouyi.com/xoql_supportedFieldTypesInAggregateFunctions.html)。
+ 当使用 AVG 函数时，也可以不使用 GROUP BY 子句。

例如，查询订单对象上，总金额的平均值。

```plain
SQL

SELECT AVG(amount)
FROM _order
```

+ 如果在使用 AVG 函数的同时，还查询了计算平均值字段以外的其他字段，则必须使用 GROUP BY 函数。

例如，查询订单对象上，各客户的 ID 和总金额的平均值。

```plain
SQL

SELECT accountId, AVG(amount)
FROM _order
GROUP BY accountId
```

## COUNT
返回满足查询条件的记录数（字段为空的记录不会计入）。

例如，查询客户对象上，各所有人负责的客户数量。

```plain
SQL

SELECT ownerId, COUNT(accountName)
FROM account
GROUP BY ownerId
```

目前**不支持**以下查询方式：

```plain
SQL

SELECT COUNT()
FROM account
```

## COUNT_DISTINCT
返回满足查询条件的记录数（去重后的记录数）。

例如，查询销售线索对象上的公司数量。

```plain
SQL

SELECT COUNT_DISTINCT(companyName)
FROM lead
```
