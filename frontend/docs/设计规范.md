## 一、色彩系统

### 1.1 主色

| 用途 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 主色 Primary | `#0071E3` | `#0A84FF` |
| 主色（强调） | `#0071E3` | `#FF2D55`（按钮等） |
| 次要色 Secondary | `#86868B` | - |
| 深色文字 | `#1D1D1F` | `#F5F5F7` |

### 1.2 背景色

| 层级 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 画布 Canvas | `#F5F5F7` | `#000000` |
| 表面 Surface | `#FFFFFF` | `#1C1C1E` |
| 次级表面 | `#2C2C2E` | - |
| 毛玻璃背景 | `white/70`、`white/60` | `#1C1C1E/70`、`#1C1C1E/60` |

### 1.3 语义色（标签/状态）

| 类型 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 蓝色 Blue | `blue-50` / `blue-600` | `blue-900/20` / `blue-400` |
| 绿色 Green | `green-50` / `green-600` | `green-900/20` / `green-400` |
| 橙色 Orange | `orange-50` / `orange-600` | `orange-900/20` / `orange-400` |
| 红色 Red | `red-50` / `red-600` | `red-900/20` / `red-400` |
| 靛蓝 Indigo | `indigo-50` / `indigo-600` | `indigo-900/20` / `indigo-400` |
| 灰色 Gray | `gray-50` / `gray-600` | `white/10` / `gray-400` |

### 1.4 边框色

- 浅色：`gray-200/50`、`gray-100`、`gray-100/50`
- 深色：`white/10`、`white/5`

## 二、字体与排版

### 2.1 字体族

```css
font-family: -apple-system, BlinkMacSystemFont, 'SF Pro Text', Segoe UI, Roboto, Helvetica, Arial, sans-serif;
```

- 全局开启 `antialiased`、`grayscale` 抗锯齿

### 2.2 字号层级

| 用途 | 类名 | 说明 |
|------|------|------|
| 页面标题 | `text-2xl` / `text-xl` | 24px / 20px，`font-bold` |
| 区块标题 | `text-lg` / `text-base` | 18px / 16px，`font-bold` / `font-semibold` |
| 正文 | `text-sm` | 14px |
| 辅助文字 | `text-xs` | 12px |
| 极小文字 | `text-[10px]` / `text-[9px]` | 10px / 9px |

### 2.3 字重

- `font-bold`：标题、强调
- `font-semibold`：副标题、表格表头
- `font-medium`：正文强调
- `font-mono`：金额、编号、时间戳

### 2.4 字间距

- `tracking-tight`：大标题
- `tracking-wider`：表头、标签
- `tracking-widest`：分组标签（uppercase）

| 尺寸 | 类名 | 典型用途 |
|------|------|----------|
| 小 | `rounded-lg` (8px) | 按钮、输入框 |

## 五、间距规范

### 5.1 基础单位

- 以 Tailwind 的 4px 为基准（`1` = 4px）

### 5.2 常用间距

| 场景 | 类名 | 说明 |
|------|------|------|
| 页面内边距 | `p-6` | 24px |
| 卡片内边距 | `p-4` ~ `p-6` | 16px ~ 24px |
| 元素间距 | `gap-2` ~ `gap-6` | 8px ~ 24px |
| 区块间距 | `space-y-6` | 24px |
| 按钮内边距 | `px-4 py-2` / `px-5 py-2` | 主按钮 / Tab |

### 5.3 布局尺寸

| 元素 | 尺寸 |
|------|------|
| 顶栏高度 | `h-16` (64px) |
| 侧边栏宽度 | `w-[240px]` / 收起 `w-[88px]` |
| 侧边栏外边距 | `left-4 top-4 bottom-4` |
| Logo 高度 | `h-[18px]` |

## 六、组件规范

### 6.1 统一卡片 `.unified-card`

```
bg-white dark:bg-[#1C1C1E]
rounded-[24px]
shadow-apple
border border-gray-100/50 dark:border-white/10
overflow-hidden
transition-all duration-300
```

### 6.2 统一表格表头 `.unified-table-header`

```
bg-gray-50/80 dark:bg-white/5
text-gray-500 dark:text-gray-400
text-sm uppercase font-bold tracking-wider
border-b border-gray-100 dark:border-white/10
```

### 6.3 按钮变体

| 类型 | 类名 | 说明 |
|------|------|------|
| 主按钮 | `.unified-button-primary` | 蓝底白字，深色模式红底 |
| 次按钮 | `.unified-button-secondary` | 白底灰字，带边框 |
| 幽灵按钮 | `.unified-button-ghost` | 透明背景，悬停变色 |

### 6.4 按钮基础 `.unified-button`

```
px-4 py-2 rounded-lg font-bold text-sm
transition-all duration-200
active:scale-[0.98]
disabled:opacity-50 disabled:pointer-events-none
flex items-center justify-center gap-2
```

### 6.5 标签 `.unified-tag`

| 变体 | 用途 |
|------|------|
| unified-tag-blue | 默认、待处理 |
| unified-tag-green | 成功、已完成 |
| unified-tag-orange | 警告、进行中 |
| unified-tag-red | 错误、取消 |
| unified-tag-gray | 中性 |
| unified-tag-indigo | 特殊状态 |

### 6.6 图标

- 统一使用 `lucide-react`
- 默认尺寸：`w-5 h-5`（20px）
- 小图标：`w-4 h-4`（16px）

## 七、弹窗与抽屉

### 7.1 遮罩

- 全屏：`fixed inset-0`
- 背景：`bg-black/40` ~ `bg-black/60`
- 模糊：`backdrop-blur-sm`
- 层级：`z-[500]`（通过 ModalPortal 渲染到 `#modal-root`）

### 7.2 弹窗

- 居中：`flex items-center justify-center`
- 内边距：`p-4` ~ `p-6`
- 圆角：`rounded-3xl`（桌面端）
- 最大尺寸：`max-w-5xl`、`max-h-[90vh]`

### 7.3 抽屉

- 位置：右侧滑入
- 宽度：`max-w-md` ~ `max-w-[480px]`
- 动画：`animate-drawer-enter` / `animate-drawer-exit`
- 遮罩动画：`animate-backdrop-enter` / `animate-backdrop-exit`

### 7.4 动画时长

| 类型 | 类名 | 时长 |
|------|------|------|
| 页面进入 | `animate-page-enter` | 0.2s |
| 弹窗进入 | `animate-modal-enter` | 0.4s |
| 抽屉进入 | `animate-drawer-enter` | 0.38s |
| 抽屉退出 | `animate-drawer-exit` | 0.26s |
| 遮罩进入 | `animate-backdrop-enter` | 0.28s |
| 遮罩退出 | `animate-backdrop-exit` | 0.22s |

### 7.5 缓动曲线

- 通用：`cubic-bezier(0.16, 1, 0.3, 1)`
- 抽屉：`cubic-bezier(0.22, 1, 0.36, 1) both;`
- 退出：`cubic-bezier(0.4, 0, 1, 1) both;`

---

## 八、滚动条

### 8.1 默认样式

- 宽度/高度：6px
- 轨道：透明
- 滑块：`rgba(0,0,0,0.1)`，悬停 `0.2`
- 深色模式：`rgba(255,255,255,0.15)`，悬停 `0.25`
- 圆角：10px

### 8.2 隐藏滚动条

- 类名：`.no-scrollbar`
- 自定义滚动：`.custom-scrollbar`

---

## 九、深色模式

- 切换方式：`class`（在 `html` 上添加 `.dark`）
- 持久化：`localStorage` 的 `theme`
- 默认：跟随系统 `prefers-color-scheme: dark`
- 过渡：`transition-colors duration-300`

---

## 十、响应式断点

| 断点 | 前缀 | 典型用途 |
|------|------|----------|
| 默认 | - | 移动端 |
| sm | `sm:` | 640px+ |
| md | `md:` | 768px+ |
| lg | `lg:` | 1024px+ |

- 侧边栏：`lg:hidden` 移动端隐藏、`lg:flex` 桌面端显示
- 顶栏 Tab：`hidden md:flex` 桌面端显示
- 内容区：`grid-cols-1 lg:grid-cols-3` 等

---

## 十一、交互规范

- 悬停：`hover:bg-gray-100`、`hover:scale-105` 等
- 按下：`active:scale-[0.98]`
- 禁用：`disabled:opacity-50 disabled:pointer-events-none`
- 选中高亮：`selection:bg-blue-500/30`

---

## 十二、无障碍

- 图标：`aria-label` 等语义化属性
- 焦点：`focus:ring-0` 等自定义焦点样式
- 对比度：主色与背景对比符合 WCAG 标准
