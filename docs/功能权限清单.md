# 业务平台 — 功能权限清单

> 文档生成日期：2026-03-27
>
> 权限树采用四级结构：**一级（模块组）→ 二级（菜单/子模块）→ 三级（功能分类）→ 四级（权限点）**
>
> 对于权限点较少的子模块，可省略三级分类，直接挂载权限点。
