## 1. 产品概述

WPS 业务平台是面向 WPS 内部销售与商务团队的综合管理系统，覆盖从线索获取、商机跟进、订单创建与审批、备货交付到客户档案管理的全业务链路。系统支持亮色/暗色主题切换、全屏模式、响应式布局，并内置基于角色的权限控制体系。

**核心价值**

- 统一管理订单全生命周期（创建→审批→备货→发货→交付→回款）
- 打通客户、商机、渠道、商品多维度数据关联
- 支持多角色协同（销售、商务、财务、物流、管理员）
- 提供商品目录展示与销售商品配置能力

## 3. 导航与布局

### 顶部导航栏

顶部栏固定在页面顶部，包含以下区域：

| 区域 | 内容 |
|------|------|
| 左侧 | WPS Logo + 平台名称"业务平台" |
| 中部 | 一级导航 Tab（根据权限动态显示） |
| 右侧 | 全屏切换、深色/浅色主题切换、消息通知、当前用户头像 |

**一级导航 Tab（按权限显示）**

| Tab 名称 | 路径 | 所需权限 |
|----------|------|----------|
| 数据看板 | `/` | `dashboard_view` |
| 订单中心 | `/orders` | `order_view_*` / `customer_view` 等 |
| 商品中心 | `/product-center` | `product_view` / `merchandise_view` |
| 线索中心 | `/leads` | `leads_view` |
| 运营中心 | `/wps-ops` | `wps_ops_view` |
| 系统配置 | `/organization` | `admin_view` / `user_manage` 等（仅 Admin） |

### 侧边栏

侧边栏随选中的一级 Tab 联动切换，支持展开/收起（收起后仅显示图标）。

**订单中心侧边菜单**：订单中心、商机信息、客户信息、渠道管理

**商品中心侧边菜单**：商品展示、商品列表

**系统配置侧边菜单**：组织架构、用户管理、角色管理

**运营中心侧边菜单**（规划中）：指标看板、企业管理、CRM、订单管理、运营管理、策略管理、企业后台菜单管理、企业专属客户端、数据统计、企业画像、效率平台、日志管理、升级隔离、系统设置、数据迁移、连接器等

### 用户切换

顶部右侧用户头像点击后展开下拉菜单，支持在 mock 用户列表中快速切换当前操作身份（用于演示不同角色的权限差异）。

### 主题与全屏

- **主题**：支持亮色/暗色切换，状态持久化至 `localStorage`
- **全屏**：调用浏览器原生全屏 API

## 5. 订单中心

### 5.1 订单列表

路径：`/orders`

#### 列表字段

| 字段名 | 说明 |
|--------|------|
| 订单编号 | 格式：`S` + 6位日期(YYMMDD) + 12位随机数，点击跳转详情 |
| 商品信息 | 商品名称 + SKU 蓝色标签（最多展示2条） |
| 所属部门 | 显示末级部门名（加粗）+ 上级部门（浅色） |
| 销售负责人 | 头像 + 姓名（去除括号内容）+ 邮箱 |
| 商务经理 | 同上 |
| 订单来源 | 枚举标签：后台下单 / 渠道下单 / 官网下单 / 第三方下单 / 客户续费 |
| 订单类型/买方类型 | 枚举标签 |
| 订单状态 | 带颜色圆点的状态标签 |
| 付款状态 | 已支付 / 待支付 |
| 备货状态 | 已完成 / 已发货 / 待发货 / 备货中 / 待处理 |
| 客户名称 | — |
| 操作 | 根据订单状态显示"去审批"等操作按钮（无状态时为空） |

#### 列表交互

- **固定列**：订单编号列（最左）和操作列（最右）在横向滚动时保持固定（`position: sticky`）
- **表头固定**：下拉滚动时表头保持可见
- **点击订单编号**：进入订单详情（仅编号可点击，行级点击已移除）
- **水平滚动阴影**：固定列带有边缘阴影以提示滚动

#### 搜索

搜索框位于"订单中心"标题右侧，支持通过左侧下拉切换搜索维度：

| 搜索维度 | 说明 |
|----------|------|
| 订单编号 | 精确/模糊匹配订单 ID |
| 客户名称 | 模糊匹配客户公司名 |
| 买方名称 | 模糊匹配买方名称 |

#### 筛选与排序

支持多维度筛选（状态、来源、时间范围等），支持列显示/隐藏控制。

#### 分页

- 默认每页 20 条
- 可选每页 50 / 100 条
- 分页器选中状态使用蓝色主色调

#### 新建订单

列表右上角提供「新建订单」入口，支持填写客户、商品、买方、渠道、金额等完整订单信息。

#### 订单状态枚举

| 状态值 | 显示名称 | 说明 |
|--------|----------|------|
| `PENDING_APPROVAL` | 待审批 | 订单已提交，等待销售/商务/财务审批 |
| `PENDING_CONFIRM` | 待确认 | 审批通过，等待确认 |
| `PROCESSING_PROD` | 备货中 | 已确认，正在备货 |
| `PENDING_PAYMENT` | 待收款 | 等待客户付款 |
| `SHIPPED` | 已发货 | 已完成发货 |
| `DELIVERED` | 已完成 | 订单已交付 |
| `CANCELLED` | 已取消 | 订单已取消 |
| `REFUND_PENDING` | 退单审批中 | 退单申请已提交 |
| `REFUNDED` | 已退单 | 退单完成 |

- 表格底部汇总区：产品总金额、优惠折扣、返利折扣、优惠金额、**订单实付金额**（全部橙色）

> **商品明细详情 Drawer**  
> 点击明细编号弹出，顶部显示 **明细编码**（格式：`订单编号-001`，橙色标签）+ 商品名称 + SKU 标签。  
> 内容包含：授权信息、技术规格与安装包（字段：编号、安装包名称、发布平台、操作系统、CPU）、生命周期与续费信息。

**客户信息卡片（中型，占 2/3）**

卡片标题：`Building 图标（蓝色）+ 客户信息`

字段（双列布局）：

| 字段 | 说明 |
|------|------|
| 客户名称 | 蓝色可点击，跳转至客户详情 |
| 客户类型 | 央企/SMB/政府/教育/合作伙伴 |
| 行业条线 | — |
| 所在地区 | 省份 / 城市 / 区县（合并展示） |
| 报备标签 | — |
| 是否授权覆盖客户 | — |

**交易方信息卡片（中型，占 2/3）**

卡片标题：`Users 图标（靛蓝）+ 交易方信息`

字段（双列布局）：买方名称 | 直接下单渠道 | 终端渠道 | 渠道是否提供服务

**商机信息卡片（小型，占 1/3）**

卡片标题：`Target 图标（橙色）+ 商机信息`

字段：商机名称 | CRM ID | 商机阶段 | 商机金额 | 结单日期

**开票明细卡片（小型，占 1/3）**

卡片标题：`Receipt 图标（绿色）+ 开票明细`

字段：发票抬头 | 纳税人识别号 | 发票类型

**关联原始订单卡片（小型，占 1/3）**

卡片标题：`History 图标（紫色）+ 关联原始订单`

字段：SMS原订单编号 | SaaS原订单编号（等宽字体）

#### 字段展示规范（所有卡片统一）

| 元素 | 规格 |
|------|------|
| 卡片标题字体 | 18px，`font-semibold`，深色 |
| 卡片标题图标 | 20×20px，各卡片使用专属颜色 |
| 字段名 | 14px，灰色（`text-gray-400`），**右对齐** |
| 字段值 | 16px，`font-medium`，深色 |
| 标签与值间距 | 32px（gap-8） |
| 字段行间距 | 10px 上下（py-2.5） |
| 行分割线 | 浅灰 divide-y |

#### 审批流程 Tab

展示三级审批链：销售审批 → 商务审批 → 财务审批。每级显示审批人、审批结果、时间戳。当前待审批节点支持"通过"操作。

#### 备货管理 Tab

包含四个备货子步骤：

1. **授权码确认**：确认电子授权内容
2. **安装包就绪**：查看关联安装包（字段：编号、安装包名称、发布平台、操作系统、CPU），点击确认就绪
3. **CD 刻录**：确认刻录完成
4. **快递单确认**：填写快递服务商和物流单号

所有步骤完成后解锁"确认发货"操作。

#### 收款管理 Tab

显示付款记录（金额、日期、银行、账号、流水号），支持登记新收款记录。

#### 验收管理 Tab

支持一次性验收和分阶段验收两种模式，可配置验收联系人（姓名、电话、邮箱、方式：远程/现场）和验收阶段（比例、金额）。

#### 退单流程

满足以下条件时，顶栏显示"退单"按钮：订单已付款 且 状态不为退单中/已退单/已取消。退单需填写退单原因和金额，提交后进入退单审批流程。
