# AI 业务助手 — 操作手册

> 适用对象：销售人员、商务人员、业务管理者
>
> 适用平台：WPS 365 业务平台

---

## 一、什么是 AI 业务助手？

AI 业务助手是内嵌在 WPS 365 业务平台中的智能问答工具。你可以用**自然语言**直接向它提问，它会自动理解你的意图，结合**当前页面的业务数据**，给出专业的分析结果。

**它能做什么：**

- 帮你快速识别订单风险、价格异常、回款问题
- 判断某个订单是新购还是续费/增购/升级
- 一键生成客户360画像，了解客户全貌
- 自动汇总业绩数据，生成业务简报

**它不会做什么：**

- 不会编造数据——所有分析都基于系统中的真实数据
- 不会回答与业务无关的问题（如天气、新闻等）
- 不会跨权限访问数据——每个分析只使用必要的数据范围

---

## 二、如何打开 AI 助手

1. 登录 WPS 365 业务平台
2. 在页面**右下角**，可以看到一个紫色的 **"AI 助手"** 按钮
3. 将鼠标移到按钮上，会滑出完整的按钮文字
4. **点击按钮**，弹出 AI 对话框

> 提示：AI 助手按钮在平台的所有页面都可以看到，包括数据看板、订单管理、客户信息等。

---

## 三、AI 助手的四大技能

### 1. 风险管控

**适用场景：** 审核订单时快速排查风险点

**能帮你分析：**
- 订单状态是否异常（如长期未付款、异常取消）
- 成交价格与建议售价是否有较大偏差
- 回款进度是否正常，是否存在逾期风险
- 订单信息与合同条款是否一致
- 订单中的客户信息与 CRM 记录是否匹配

**你可以这样问：**

| 场景 | 问法示例 |
|------|----------|
| 检查当前订单 | "这个订单有没有风险？" |
| 价格审核 | "价格是否异常？和建议售价差多少？" |
| 回款检查 | "回款情况怎么样？有没有逾期？" |
| 合同比对 | "订单和合同金额是否一致？" |
| 批量排查 | "哪些订单存在风险？"（在订单列表页提问） |

---

### 2. 续费分析

**适用场景：** 判断订单的购买性质，梳理续费关系链

**能帮你判断：**
- 当前订单是**新购**、**续费**、**增购**还是**升级**
- 续费关系链：原始订单 → 续费订单的关联
- 同一客户在同一产品线上的历史购买记录
- 授权到期时间，提前提醒续费机会

**你可以这样问：**

| 场景 | 问法示例 |
|------|----------|
| 判断购买性质 | "这个订单是新购还是续费？" |
| 查看续费链 | "续费关系是什么？关联了哪些历史订单？" |
| 续期判断 | "这是续期还是增购？" |
| 升级分析 | "客户是不是从标准版升级到了旗舰版？" |

---

### 3. 客户360

**适用场景：** 拜访客户前快速了解客户全貌，制定销售策略

**能帮你查看：**
- 客户基本档案：公司名称、行业、类型、等级、区域
- 历史订单：购买次数、累计金额、产品偏好
- 商机情况：进行中的商机、预期收入
- 合同状态：合同签订及验证情况

**你可以这样问：**

| 场景 | 问法示例 |
|------|----------|
| 了解客户全貌 | "这个客户整体情况怎么样？" |
| 查看历史订单 | "客户历史订单有多少？买了什么产品？" |
| 客户档案 | "客户档案信息？行业、等级？" |
| 客户价值 | "这个客户累计贡献了多少收入？" |

---

### 4. 业务总结

**适用场景：** 周会/月会前快速生成业务简报，了解业绩概况

**能帮你生成：**
- 按时间段的销售额、订单量统计
- 客户分布：类型占比、等级分布
- 商机漏斗：各阶段数量和转化率
- 业绩趋势和对比分析
- 结构化的业务简报

**你可以这样问：**

| 场景 | 问法示例 |
|------|----------|
| 月度总结 | "帮我总结一下本月业绩" |
| 业务简报 | "帮我做一份业务简报" |
| 数据统计 | "目前总共有多少订单？总金额多少？" |
| 客户分析 | "客户类型分布情况？" |
| 商机概览 | "商机各阶段分别有多少？" |

## 四、页面感知——在哪里提问，分析哪里的数据

AI 助手会**自动识别你当前所在的页面**，只分析与当前页面相关的数据，确保回答精准且高效。

### 不同页面的数据范围

| 你所在的页面 | AI 分析的数据范围 | 对话框顶部显示 |
|---|---|---|
| **数据看板**（首页） | 全局数据概览统计 | 数据看板 |
| **订单管理**（列表页） | 系统中的订单数据 | 订单列表 |
| **某个订单的详情页** | 仅当前这一笔订单 + 关联的客户、合同、商机 | 订单详情 ORD-xxx |
| **客户信息**（列表页） | 系统中的客户数据 | 客户列表 |
| **某个客户的详情页** | 仅当前这个客户 + 其所有订单、商机、合同 | 客户详情 C-xxx |
| **商机信息** | 系统中的商机数据 | 商机列表 |
| **合同信息** | 系统中的合同数据 | 合同列表 |

### 实际操作举例

**场景一：审核某笔订单**
1. 进入「订单管理」，点击某笔订单进入详情页
2. 点击右下角「AI 助手」按钮
3. 对话框顶部显示「订单详情 ORD-xxx」——说明 AI 已锁定当前订单
4. 直接问"这个订单有风险吗？"——AI 只分析这一笔订单的数据

**场景二：拜访前了解客户**
1. 进入「客户信息」，点击目标客户进入详情页
2. 打开 AI 助手
3. 对话框顶部显示「客户详情 C-xxx」
4. 问"客户整体情况怎么样？"——AI 自动汇总该客户的档案、历史订单、商机

**场景三：准备周会简报**
1. 回到「数据看板」首页
2. 打开 AI 助手
3. 问"帮我做一份本周业务简报"——AI 基于全局数据生成汇总

## 五、快捷提问

打开 AI 助手后，如果还没开始对话，底部会显示**快捷提问按钮**。这些按钮会根据你当前所在的页面自动变化。

- 在**订单详情页**：「这个订单有风险吗？」「这是新购还是续费？」「关联客户情况」「订单信息总结」
- 在**订单列表页**：「哪些订单有风险？」「订单数据总结」「回款情况分析」「续费订单有哪些？」
- 在**客户详情页**：「这个客户整体情况？」「历史订单总结」「客户有续费吗？」「客户风险分析」
- 在**数据看板**：「本月业绩总结」「订单风险概览」「客户整体情况」「业务简报」

> 点击任意快捷提问按钮，问题会自动填入输入框，再点发送即可。

## 六、对话操作说明

### 基本操作

| 操作 | 方法 |
|------|------|
| 发送消息 | 输入问题后，点击发送按钮 或 按 **Enter** 键 |
| 换行输入 | 按 **Shift + Enter** |
| 清空对话 | 点击对话框顶部的 🗑 清空按钮 |
| 关闭对话框 | 点击 ✕ 关闭按钮，或点击对话框外的灰色区域 |

### 对话界面说明

- **顶部栏**：显示 AI 助手图标、当前所在页面、已匹配的技能名称
- **消息区域**：你的消息显示在右侧（蓝色气泡），AI 回复显示在左侧（灰色气泡）
- **技能标签**：AI 每次回复上方会显示使用了哪个技能（如"风险管控""客户360"）
- **加载状态**：AI 正在思考时，会显示跳动的圆点动画
- **底部输入框**：支持多行输入，输入框会自动扩高

## 七、常见问题

### Q1：AI 助手回答"请更具体地描述您的问题"怎么办？

这说明你的问题比较模糊，AI 无法确定该用哪个技能来回答。请尝试：
- 加入具体的业务方向关键词，如"订单风险""客户情况""续费""业绩总结"
- 使用快捷提问按钮，它们已经为你预设了精准的问题

### Q2：AI 助手能看到哪些数据？

AI 助手只能看到你在当前页面上下文中相关的数据。例如在订单详情页，它只能看到当前这笔订单及其关联的客户、合同信息，看不到其他订单。

### Q3：AI 回答的数据准确吗？

AI 助手的分析严格基于系统中的真实数据，不会编造。但 AI 的分析解读（如风险判断、趋势预测）是辅助参考，最终决策请结合实际业务情况。

### Q4：我能把 AI 的分析结果给客户看吗？

AI 分析结果为内部参考，包含系统业务数据，请勿直接转发给外部客户。

### Q5：AI 助手为什么有时候回复比较慢？

AI 回答需要调用云端大模型进行分析，通常需要 3-10 秒。如果网络较慢或数据量较大，等待时间会稍长。

### Q6：切换到其他页面后，AI 助手的上下文会变吗？

会的。AI 助手每次提问时都会自动检测你当前所在的页面，数据范围会随之变化。如果你从订单详情页切换到客户列表页，新的提问将基于客户列表的数据来回答。
