## 一、平台概述

WPS 365 业务平台是面向销售、商务、运营团队的一体化业务管理系统，覆盖从线索获取到订单交付的完整业务链条。

### 核心业务流程

```
线索 → 商机 → 客户 → 新建订单 → 审批 → 确认 → 备货 → 发货 → 验收 → 交付完成
```

### 功能模块一览

| 模块 | 主要功能 | 面向角色 |
|------|----------|----------|
| 数据看板 | 业绩概览、趋势图表、动态播报 | 全员 |
| 订单中心 | 订单全生命周期管理 | 销售、商务 |
| 客户管理 | 客户档案、联系人、企业管理 | 销售 |
| 商机管理 | 商机跟进、阶段推进 | 销售 |
| 产品中心 | 产品目录、SKU、报价 | 销售、商务 |
| 渠道管理 | 渠道商信息管理 | 销售、商务 |
| 业绩管理 | 业绩核算、归属查询 | 销售、管理者 |
| 运营中心 | 备货处理、指标看板、企业管理 | 运营 |
| 系统管理 | 用户、角色、权限、组织架构 | 管理员 |
| AI 助手 | 智能问答、风险分析、业务总结 | 全员 |

## 二、登录与界面导航

### 2.1 整体布局

平台采用"顶部导航 + 左侧菜单 + 标签页"的经典布局：

- **顶部导航栏**：左侧为 WPS Logo 和平台名称，中间为一级功能入口（数据看板、订单中心、产品中心等），右侧为全屏切换、主题切换（明/暗）、产品手册、用户菜单
- **左侧菜单栏**：根据选中的顶部导航项动态切换，可折叠收起
- **标签页栏**：浏览过的页面会以标签形式保留，支持右键关闭、关闭其他、关闭全部
- **右下角浮动按钮**：AI 助手入口、意见反馈

### 2.2 一级导航入口

| 导航项 | 包含的二级菜单 |
|--------|---------------|
| 数据看板 | 数据看板（首页） |
| 订单中心 | 订单管理、客户信息、商机信息、合同信息、汇款管理、发票管理、授权信息、交付信息 |
| 产品中心 | 产品目录、产品列表、产品政策、组件池、安装包管理、产品授权模板、属性配置、建议售价、渠道价格 |
| 业绩中心 | 业绩管理 |
| 渠道中心 | 渠道管理 |
| 线索中心 | 线索管理 |
| 运营中心 | 备货处理、指标看板、企业管理 |
| 系统配置 | 组织架构、用户管理、角色管理、授权类型管理、产品发货方式配置、销售组织配置 |

> 注意：各导航项和菜单的可见性由你的角色权限决定，部分功能可能不可见。

### 2.3 主题切换

点击顶部右侧的太阳/月亮图标可切换**浅色/深色**主题，设置会自动保存。

### 2.4 产品手册

点击顶部的"产品手册"图标可打开内置的使用说明文档，以抽屉形式展示。

## 三、数据看板

数据看板是登录后的默认首页，提供业务全局概览。

### 3.1 核心指标卡片

页面顶部展示四个核心 KPI 卡片：

| 指标 | 说明 |
|------|------|
| 总销售额 | 累计销售金额（元） |
| 活跃客户 | 当前活跃客户数量 |
| 订单总量 | 累计订单数量 |
| 商机转化率 | 商机转化为订单的比率 |

每个卡片右上角显示环比趋势（↑ 增长 / ↓ 下降）。

### 3.2 图表区域

- **销售趋势图**：折线/面积图展示月度销售额和订单量趋势，可切换"最近7个月"或"最近12个月"
- **订单状态分布**：环形图展示各状态订单的占比（已交付、待审批、待付款、已取消）
- **热门商品排行**：横向柱状图展示最畅销的产品排名

### 3.3 近期业务动态

底部列表滚动展示最近的业务事件，包括商机创建、订单审批、回款确认、线索同步、授权码生成等。

## 四、订单中心

订单中心是平台的核心模块，覆盖订单从创建到交付的全生命周期。

### 4.1 订单管理（列表页）

#### 订单状态流转

订单按以下流程推进：

```
草稿 → 待审批 → 待确认 → 备货中 → 待支付 → 已发货 → 已完成
                                                     ↘ 退款中 → 已退款
                                          ↘ 已取消
```

#### 状态管道卡片

页面顶部以卡片形式展示各状态的订单数量，点击即可快速筛选：

| 状态 | 说明 |
|------|------|
| 草稿 | 已保存但未提交的订单 |
| 待审批 | 已提交，等待销售/商务/财务审批 |
| 待确认 | 审批通过，等待订单确认 |
| 备货中 | 确认后进入备货流程（含授权确认、安装包核验、快递单填写、光盘刻录四个子步骤） |
| 待支付 | 备货完成，等待客户付款 |
| 已发货 | 已发出物流 |
| 已完成 | 客户验收通过，订单完成 |
| 退款中 | 客户申请退款，处理中 |
| 已退款 | 退款已完成 |
| 已取消 | 订单已取消 |

#### 搜索与筛选

**快捷搜索**：选择搜索字段（订单编号 / 客户名称 / 买方名称 / 产品名称），输入关键词搜索。

**快捷筛选**：
- 订单来源：后台下单、渠道下单、官网、三方平台、客户续费
- 提单日期：选择起止日期
- 应付金额：设置最小/最大金额范围

**高级筛选**：点击"高级筛选"按钮打开筛选面板，可组合以下条件：

| 筛选维度 | 筛选方式 |
|----------|----------|
| 订单来源 | 多选 |
| 订单类型 | 多选 |
| 商务 | 选择商务人员 |
| 订单状态 | 多选 |
| 付款状态 | 单选 |
| 备货状态 | 单选 |
| 提单时间 | 时间段 / 精确时间点 |
| 发货时间 | 时间段 / 精确时间点 |
| 订单应付金额 | 金额范围 |

点击"查询"应用筛选，点击"重置"清空所有条件。

#### 列表操作

| 操作 | 说明 |
|------|------|
| 点击订单编号 | 进入订单详情页 |
| 点击客户名称 | 跳转到客户详情页 |
| 复制订单号 | 点击行操作复制订单编号到剪贴板 |
| 复制表格 | 将当前可见表格数据复制为 TSV 格式 |
| 列配置 | 点击列配置按钮自定义显示哪些列 |
| 批量确认 | 勾选多个"待确认"订单，点击批量确认 |
| 批量发货 | 勾选多个"备货中"订单，点击批量发货 |

### 4.2 新建订单

点击"新建订单"按钮，进入四步向导式创建流程。

#### 第一步：订单类型

| 填写项 | 说明 |
|--------|------|
| 买方类型 | 直签 / 渠道 / 自成交 / 兑换码（必选） |
| 订单来源 | 后台下单 / 渠道下单 / 官网 / 三方平台 / 客户续费 |
| 关联合同 | 可选择关联已有合同 |
| 订单备注 | 备注信息 |

> 如果是续费订单，会自动标记"续费模式"，并关联原始订单。

#### 第二步：客户信息

| 填写项 | 说明 |
|--------|------|
| 客户 | 选择或搜索已有客户 |
| 关联商机 | 选择是否关联商机，如"是"则选择具体商机 |
| 渠道 | 买方类型为"渠道"时需选择渠道商 |
| 销售 / 商务 | 选择负责的销售人员和商务人员 |
| 制单人 | 当前操作人 |
| 企业 ID | 关联客户的企业租户 |
| 采购联系人 / IT联系人 | 从客户联系人中选择，也可新建 |

#### 第三步：产品配置

| 填写项 | 说明 |
|--------|------|
| 产品类目 | 按类目级联选择产品 |
| SKU | 选择具体规格 |
| 定价项 | 选择报价方案 |
| 数量 | 购买数量 |
| 激活方式 | 序列号 / 在线 / 加密锁 / 账号 等 |
| 购买性质 | 新购 / 续费 / 增购 / 升级 |
| 续费子类型 | 续费时需选择：续期 / 增购 / 升级 |
| 授权期限 | 授权开始/结束时间 |
| 被授权方 / 企业 | 关联的企业信息 |
| 安装包类型 | 通用 / 定制 |

> 可添加多个产品行。每行可单独设置价格、授权信息和议价。

#### 第四步：交付信息

| 填写项 | 说明 |
|--------|------|
| 发票信息 | 发票类型（增值税普通/专用）、税号、抬头、银行等 |
| 支付方式 | 微信 / 支付宝 / 银行转账 |
| 支付条款 | 账期约定 |
| 发货方式 | 线上 / 线下 / 混合 |
| 收货信息 | 收货方、公司、地址、电话、邮箱 |
| 验收方式 | 一次性验收 / 分期验收（可按比例设置多个阶段） |
| 结算方式 | 现结 / 信用 / 预付 |
| 分期计划 | 信用结算时设置分期付款计划 |

完成四步后点击"提交订单"，订单进入"待审批"状态。也可随时点击"保存草稿"暂存。

### 4.3 订单详情页

点击订单编号进入详情页，页面顶部显示：

- **订单编号**（可复制）、提单时间
- **标签**：买方类型、订单来源
- **状态徽标**：订单状态 / 付款状态 / 备货状态
- **负责人**：销售、商务、制单人信息

#### 工作流步进器

详情页核心是**工作流步进器**，展示订单当前处于哪个环节，点击对应步骤可执行操作：

| 步骤 | 操作 | 说明 |
|------|------|------|
| 支付 | 确认收款 | 记录客户付款信息 |
| 审批 | 销售审批 → 商务审批 → 财务审批 | 三级审批，每级可通过或驳回并填写意见 |
| 确认 | 订单确认 | 确认订单信息无误，进入备货 |
| 备货 | 四个子步骤依次完成 | 授权确认 → 安装包核验 → 快递单填写 → 光盘刻录 |
| 发货 | 填写物流信息 | 填写承运商、运单号，确认发货 |
| 验收 | 验收确认 | 按验收计划执行一次性/分期验收 |

> 自成交订单流程简化：支付完成后直接进入"交易完成"。

#### 详情页标签页

| 标签页 | 内容 |
|--------|------|
| 订单信息 | 产品明细表、价格汇总（优惠/返利/折算/应付/实付）、商机信息、客户信息、交易双方、合同信息、结算方式、验收信息 |
| 序列号 | 授权序列号列表（序列号、来源、生成方式、类型、状态、时间） |
| 订单交付 | 交付列表、电子授权预览、支持授权变更和二次交付申请 |
| 发货信息 | 物流/邮件发货详情 |

#### 订单操作按钮

| 按钮 | 说明 |
|------|------|
| 编辑订单 | 仅草稿状态可用，重新进入编辑 |
| 快照 | 查看订单历史快照 |
| 操作记录 | 查看所有操作日志（谁在什么时间做了什么） |
| 退单 | 已付款的订单可发起退款申请 |

### 4.4 合同信息

管理与订单关联的合同。

**列表字段**：合同名称/编码、外部合同编码、合同类型、甲方、乙方、核查状态、核查备注。

**合同类型**：渠道最终用户合同、直销合同、框架合同、服务合同、补充协议。

**核查状态流转**：

```
待商务核查 → 待核查 → 核查中 → 核查通过 / 核查不通过
```

支持搜索（合同名称/编码/甲方）和分页。

### 4.5 汇款管理

查看和管理汇款记录。

**列表字段**：汇款单号、ERP 单据号、银行流水号、类型（渠道/客户）、汇款方、付款方式、金额、收款方、支付时间。

**筛选**：关键字搜索、支付时间范围、类型筛选（渠道/客户）、付款方式筛选（电汇/网银/支票/现金）。

页面顶部显示筛选后的**合计金额**。

### 4.6 发票管理

管理开票申请。

**列表字段**：发票编号、发票抬头、金额、申请时间、申请类型、状态、关联订单。

**申请类型**：开票 / 红冲 / 换票。

**发票状态**：待开票 → 开票中 → 已开票 / 已驳回 / 已取消。

### 4.7 授权信息

查看产品授权记录。

**列表字段**：授权编号、订单编号、被授权方、客户名称、产品名称、产品编码、授权期间（起止时间）、服务期间（起止时间）。

支持按授权编号/订单编号/客户名称/产品名称搜索，支持按被授权方地区筛选。

### 4.8 交付信息

查看订单交付记录。

**列表字段**：交付编号、交付类型、订单编号、数量、授权类型、被授权方、客户名称、授权编码、授权/服务期间。

支持搜索和按交付类型、授权类型筛选。

## 五、客户管理（CRM）

### 5.1 客户列表

管理所有客户档案。

**列表字段**：客户编号、客户名称、关联企业 ID、客户类型、行业条线、客户级别、省份、客户属性。

**搜索**：按客户编号/客户名称/关联企业 ID 搜索。

**高级筛选**：

| 筛选维度 | 选项示例 |
|----------|----------|
| 客户类型 | 学校、中央党政机关、地方党政机关、央企、地方国企、民企、金融… |
| 行业条线 | 政务特种、大客民企、企业区域金融、教育业务、医疗行业… |
| 客户级别 | 一级/中央级、二级/省级、三级/市级、四级/县级… |
| 省份 | 全国所有省市自治区 |
| 客户属性 | 2B、2C、2B&2C |

点击客户编号可进入客户详情页。

### 5.2 客户详情页

客户详情页分为六个标签页：

#### 客户信息

展示客户的完整档案：
- 基本信息：编号、名称、报备标签、客户属性、类型、行业条线、级别
- 地区信息：国家、省、市、区、地址
- 归属信息：所有人、所在部门、经销商名称
- 其他：公司电话、创建时间

#### 组织信息

管理客户关联的企业租户：
- 查看已关联的企业列表（企业 ID、名称、创建时间、来源）
- **关联企业**：输入已有企业 ID 进行关联
- **创建企业**：新建企业租户并自动关联到当前客户

#### 联系人信息

管理客户的联系人：
- 以卡片形式展示所有联系人（姓名、手机、邮箱、角色、是否主联系人）
- **新增联系人**：填写姓名、手机、邮箱，选择角色（采购/IT/财务/管理/其他）
- **编辑联系人**：修改联系人信息
- **设置主联系人**：每个客户只有一个主联系人

#### 收货地址

管理客户的收货地址列表，支持新增和编辑。

#### 发票信息

管理客户的开票信息：
- 发票类型：增值税普通发票 / 增值税专用发票
- 填写：税号、抬头、开户银行、账号、注册地址、注册电话

#### 银行账号

管理客户的银行账户信息，支持新增、编辑和删除。

### 5.3 商机管理

跟踪和管理所有销售商机。

**列表字段**：商机编号、名称、客户、产品信息（SKU/授权类型）、商机阶段、部门、商机金额、最终用户成交额、结单日期。

**商机阶段**（按推进顺序）：

| 阶段 | 含义 |
|------|------|
| 需求判断 | 初步接触，判断客户需求 |
| 确认商机 | 确认为有效商机 |
| 确认渠道 | 确定销售渠道方式 |
| 证实方案 | 方案验证和报价阶段 |
| 赢单 | 成功签约 |
| 输单 | 商机丢失 |

### 5.4 线索中心

管理销售线索来源。

页面展示线索概览统计卡片和线索列表，包含线索名称、公司、来源、状态、预估金额等信息。

## 六、产品中心

### 6.1 产品目录

以卡片形式浏览所有产品。

- **左侧分类树**：按产品类目层级筛选
- **状态筛选**：在架 / 下架 / 全部

## 十二、常见问题

### Q1：我看不到某些菜单或按钮？

菜单和操作按钮的可见性由你的角色权限控制。如需开通权限，请联系系统管理员。

### Q2：订单审批被驳回了怎么办？

审批驳回后，订单不会自动返回草稿。请查看驳回意见（在操作记录中），根据意见修改后重新提交。

### Q3：如何发起退款？

在订单详情页，已付款且未退款/取消的订单，点击顶部的"退单"按钮，填写退款原因和金额后提交。

### Q4：草稿订单在哪里找？

在订单管理页面，点击状态管道中的"草稿"卡片，或者在草稿箱中查看所有暂存的订单草稿。

### Q5：如何查看订单的操作历史？

在订单详情页，点击顶部的"记录"按钮，可查看该订单的完整操作日志，包括每一步的操作人、时间和备注。

### Q6：怎么给客户新增联系人？

进入客户详情页 → 切换到"联系人信息"标签 → 点击"新增联系人" → 填写信息后保存。

### Q7：如何切换深色模式？

点击顶部导航栏右侧的太阳/月亮图标即可切换。设置会自动保存到浏览器中。

### Q8：产品目录中的产品上下架是什么意思？

- **在架**：产品可在新建订单时被选择
- **下架**：产品暂停销售，新建订单时不可选
