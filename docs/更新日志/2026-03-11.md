# 业务平台更新日志 · 2026 年 3 月 11 日

---

## 订单中心

### 新建订单

- **4 步创建向导**：支持按步骤填写「订单类型 → 客户信息 → 产品配置 → 商务交付」，流程清晰、必填项校验完整。
- **订单来源与销售模式**：支持选择订单来源（后台下单 / 官网下单 / 渠道下单 / 第三方系统）及销售模式（客户直签 / 渠道代理 / 自主成交）。
- **制单人选择**：第一步可指定制单人，默认当前用户，支持在活跃用户中切换。
- **客户与商机**：选择客户后自动带出开票信息、验收联系人；支持关联企业（租户）、关联商机、销售负责人与商务负责人。
- **产品配置**：产品 → 规格 → 授权类型（如有）→ 数量与议价单价，支持多行加入清单；支持关联客户下企业（租户）。
- **商务交付配置**：开票信息、支付方式（在线支付 / 银行转账 / 货到付款）、发货方式（线上 / 线下 / 混合）、收货信息与验收计划（整体验收 / 分期验收）。
- **续费模式**：从订单详情发起续费时，自动带入原订单客户、商机、销售/商务及产品明细，并打开创建向导至产品步骤供确认与编辑。
- **权限控制**：新建订单入口受 `order_create` 权限控制；提交成功后跳转至新订单详情页，列表同步更新。

### 列表与筛选

- **多维度搜索**：支持按订单编号、客户名称、买方名称、产品名称切换搜索字段，输入即筛。
- **高级筛选**：侧边抽屉式筛选，支持多条件组合：订单来源、订单类型、商务、订单状态、付款状态、备货状态、提单时间、发货时间、订单金额等；支持多选、单选、时间段、时间点、金额范围等模式。
- **状态卡片筛选**：全部订单及 pipeline 状态（待审批、待确认、授权确认、安装包核验、快递单填写、光盘刻录、待支付、已发货、已完成）与异常状态（退款中、已退款、已取消），按权限展示并显示各状态数量。
- **重置与查询**：支持一键重置所有筛选；高级筛选中「查询」生效、「重置」清空条件。

### 表格与操作

- **列显示配置**：通过设置按钮自定义显示列（订单编号、客户、买方、产品、销售、商务、部门、来源、订单类型、提单时间、状态、付款状态、备货状态、金额、支付方式、发货方式、地址、发票、关联商机、操作等），支持保存当前配置。
- **固定列与横向滚动**：订单编号与操作列固定，表头与表体横向滚动联动，便于宽表浏览。
- **批量操作**：多选订单后底部浮动栏支持「批量确认」（待确认订单）、「批量发货」（备货中订单），含未支付订单时二次确认。
- **行内快捷操作**：待审批订单可「去审批」、待确认订单可「去确认」，点击跳转详情页并打开对应操作区。
- **用户与关联跳转**：销售/商务列支持点击打开用户详情抽屉；客户名、买方（渠道）、关联商机支持点击跳转对应详情页。
- **产品气泡**：订单多产品时展示首条 +「+N 更多」气泡，展开可查看该订单全部产品明细。

### 分页与数据

- **分页**：支持每页 20 / 50 / 100 条切换，页码与上一页/下一页；底部展示当前筛选结果总数。

## 测试与质量

- 新建订单功能于 **2026-03-11** 完成两轮测试，覆盖 4 步向导、必填校验、数据自动带出、创建成功跳转及列表更新等场景。
- 已知待优化：订单 ID 格式与现有 Mock 数据统一、授权类型未选时的提示方式、分阶段验收百分比校验的交互优化。

## 技术说明

- 订单中心主组件：`OrderManager.tsx`（列表、筛选、新建向导、批量操作、用户抽屉等）。
- 新建订单与续费入口依赖路由与 `location.state` 传递续费原单信息；权限依赖角色配置中的 `order_create`、`order_view_*` 等权限项。
